# Tutoriels
Ici vous trouverez mes Tuto d'installations d'OS pour créer des templates "matrice clonable".

Cela inclus en général:
- la création de la machine virtuelle
- l'installation de l'OS
- la configuration de base
- la préparation pour que chaque machine virtuelle puis être clonable afin d'éviter les erreurs dues aux doublons de GUID et UUID
